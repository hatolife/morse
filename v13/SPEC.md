# v13 モールス符号フラッシュカード仕様

## 目的

モールス符号そのものを、文字、符号、音の対応でフラッシュカード式に学習できるページを追加する。

既存の CW 略語・Q 符号学習ページとは対象が異なるため、既存 `#flashcard` は維持し、新しい別ページとして実装する。

## ルート

- ページ名: モールス符号フラッシュカード
- ルート: `#morse-card`
- メニュー: 既存メニューに新しい項目として追加する

## 学習対象

初期版では欧文と記号を対象にする。

### 英字

- `A` から `Z`

### 数字

- `0` から `9`

### 記号

| 表示 | モールス符号 | 備考 |
| --- | --- | --- |
| `.` | `.-.-.-` | ピリオド |
| `,` | `--..--` | カンマ |
| `:` | `---...` | コロン |
| `?` | `..--..` | 疑問符 |
| `+` | `.-.-.` | プラス |
| `-` | `-....-` | ハイフン |
| `×` | `-..-` | 乗算記号 |
| `/` | `-..-.` | スラッシュ |
| `@` | `.--.-.` | アットマーク |
| `(` | `-.--.` | 左括弧 |
| `)` | `-.--.-` | 右括弧 |
| `"` | `.-..-.` | ダブルクォート |
| `'` | `.----.` | シングルクォート |

`X` と `×` は同じ符号 `-..-` を持つ。音声を聞く試験では、同一符号のカードを同時に選択肢へ出さない。

## モード

既存の CW 略語フラッシュカードページに合わせて、3 モード構成にする。

- 一覧: 学習対象をカード表示またはリスト表示で確認する
- 学習: カードをめくり、既知/苦手を記録する
- 試験: 4 択問題で確認する

## 一覧モード

### 表示項目

- 文字
- モールス符号
- 種別
- 難易度
- 説明
- 再生操作

### 表示形式

- カード表示
- リスト表示
- 表示形式は LocalStorage に保存する

## 学習モード

### 出題形式

- 文字からモールス符号
- モールス符号から文字
- 音から文字
- 音からモールス符号

### 操作

- 再生
- 答えを見る
- 知っている
- 苦手
- 次へ
- 苦手のみ復習

## 試験モード

### 出題形式

- 文字を見てモールス符号を選ぶ
- モールス符号を見て文字を選ぶ
- 音を聞いて文字を選ぶ
- 音を聞いてモールス符号を選ぶ

### 問題数

- 10 問
- 20 問
- 50 問
- 全問

### 結果

- 正解数
- 正答率
- 合格判定
- 間違えたカード一覧
- 間違えたカードだけの復習開始

## フィルター

- 種別: 英字、数字、記号
- 難易度: 1 から 5
- 検索: 文字、モールス符号、説明

## 難易度

符号要素数と種別から初期難易度を決める。

- 1: 1 から 2 要素
- 2: 3 要素
- 3: 4 要素
- 4: 5 要素
- 5: 記号

## 状態保存

既存ページと同様に LocalStorage へ保存する。

- 既知カード
- 苦手カード
- フィルター
- 表示形式
- 一覧ソート
- 学習の出題形式
- 試験の出題形式

キーは既存 `v10.flashcard.*` と衝突しないよう、`v13.morseCard.*` を使う。

## 音声

既存 `AudioGenerator` を使う。

- 音量、周波数、WPM は設定モーダルに従う
- 一覧、学習、試験で再生できる
- 音声問題はページ表示時または再生ボタンで再生する

## データ

アプリ側の `public/morse-flashcard.tsv` に収録する。

```tsv
category	difficulty	character	morse	label	description
letter	1	E	.	E	短点1つ
symbol	5	?	..--..	question mark	疑問符
```

## 実装方針

- `v12` の `app` と `lib` 構成を踏襲する
- `morse-engine` に UI 非依存のモールスカード trainer/state を追加する
- アプリに `MorseCardView` を追加する
- 既存 `FlashcardView` は CW 略語・Q 符号用として維持する
- 既存の i18n、設定モーダル、音声、テスト構成を踏襲する
